# PM-Accelerator-Data-Science-Project-Kashyap-Dhar

This Project analyzes the "GlobalWeatherRepository.csv" file . This dataset provides Daily weather information for cities around the world. This dataset offers a comprehensive set of features that reflect the weather conditions worldwide. It includes over 40 features.
First download the dataset from kaggle from this link https://www.kaggle.com/datasets/nelgiriyewithana/global-weather-repository/code
Then download the ipynb notebook which I have uploaded and upload the dataset on Google Collab 
upload the requirements.txt file along the dataset file on google collab
download the required packages and library before running the code from requirement.txt file
